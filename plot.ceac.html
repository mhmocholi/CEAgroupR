plot.ceacR Documentation

Plot Cost-Effectiveness Acceptability Curves (CEAC)

Description

Computes and plots the probability of cost-effectiveness as a function of willingness-to-pay (lambda) using bootstrap replicates from one or several analyses. The structure of modes and faceting matches that of plot.icers.

Usage

## S3 method for class 'ceac'
plot(
  data,
  mode = "overall",
  subgroups = NULL,
  lambda_range = c(0, 1e+05),
  steps = 100,
  facet = FALSE,
  facet_by = c("vars", "levels"),
  palette = "Set2"
)

Arguments

data

A cea_results, cea_results_list, or tibble from combine_icers_results.

mode

Character. Visualization mode: one of "overall", "subgroups", or "datasets_subgroups".

subgroups

Optional character vector specifying which subgroup variables to include (for subgroup modes).

lambda_range

Numeric vector of length 2 giving the range of lambda (default = c(0, 100000)).

steps

Integer. Number of lambda points (default = 100).

facet

Logical. If TRUE, facet the plot appropriately for the mode.

facet_by

Character. For subgroup modes, determines faceting scheme: "vars" (facet by subgroup variable) or "levels" (facet by subgroup level).

palette

Color palette from RColorBrewer (default = "Set2").

Value

A ggplot object representing the cost-effectiveness acceptability curves.
