plot.icersR Documentation

Plot incremental cost-effectiveness results (ICER plane)

Description

Produces a cost-effectiveness plane using bootstrap replicates from one or several analyses. Supports flexible visualization modes for datasets and subgroups.

Usage

## S3 method for class 'icers'
plot(
  data,
  mode = "overall",
  subgroups = NULL,
  lambda = NULL,
  facet = FALSE,
  facet_by = c("vars", "levels"),
  alpha = 0.5,
  size = 1,
  palette = "Set2"
)

Arguments

data

A cea_results, cea_results_list, or tibble from combine_icers_results.

mode

Character. Visualization mode: one of "overall", "datasets", "subgroups", or "datasets_subgroups".

subgroups

Optional character vector specifying which subgroup variables to include (for modes involving subgroups).

lambda

Numeric. Willingness-to-pay threshold (optional).

facet

Logical. If TRUE, facet the plot appropriately for the mode.

facet_by

Character. For subgroup modes, determines the faceting scheme: "vars" (facet by subgroup variable) or "levels" (facet by subgroup level).

alpha

Transparency for points (default = 0.5).

size

Point size (default = 1).

palette

Color palette from RColorBrewer (default = "Set2").

Value

A ggplot object.
