icers_baseR Documentation

Bootstrap estimation of incremental cost-effectiveness ratios (ICERs)

Description

Performs nonparametric bootstrap resampling to estimate incremental cost, incremental effectiveness, ICER, and Net Monetary Benefit (NMB) between two comparison groups (e.g., usual care vs. intervention). The function also reports group-specific mean costs and effects, and optionally performs the analysis by predefined subgroups. It supports multiple datasets, allowing deterministic sensitivity analyses or scenario comparisons.

Usage

icers_base(
  data,
  group,
  cost,
  effect,
  R = 1000,
  lambda = 25000,
  ci_type = "bca",
  subgroup_vars = NULL
)

Arguments

data

A data frame or a named list of data frames containing all variables required for the analysis. All datasets in the list must share the same structure and variable names.

group

A string indicating the column name that defines the two comparison groups (reference and intervention).

cost

A string indicating the column name for total cost.

effect

A string indicating the column name for the measure of effectiveness (e.g., QALYs, life-years).

R

Integer. Number of bootstrap replications (default = 1000).

lambda

Numeric. Willingness-to-pay threshold (default = 25000).

ci_type

Character. Type of confidence interval for boot.ci ("bca", "perc", "basic", "norm").

subgroup_vars

Optional character vector. Names of subgroup variables to stratify the analysis.

Value

If a single data frame is supplied, returns a list of class cea_results containing:

  • Overall: A cea_base object with results for the full sample.

  • Subgroups: A named list of subgroup-specific results.

  • combined_replicates: A tibble combining all bootstrap samples.

If a list of data frames is supplied, returns a named list of cea_results objects (class cea_results_list) plus the element combined_replicates that aggregates results across datasets.
