combine_icers_resultsR Documentation

Combine bootstrap samples from one or several CEA analyses

Description

Combines the bootstrap replicates contained in one or more objects of class cea_results or cea_results_list into a single tidy data frame. The resulting dataset includes all bootstrap samples for the overall analysis and any available subgroup analyses. This function is intended to facilitate plotting and comparison across different analyses or datasets.

Usage

combine_icers_results(...)

Arguments

...

One or more objects of class cea_results or a single object of class cea_results_list, as returned by icers_base.

Value

A tibble containing all bootstrap replicates with the following columns:

  • dataset: dataset name (if multiple datasets were analysed).

  • level: "Overall" or name of the subgroup variable.

  • subgroup: subgroup level (if applicable).

  • replicate: bootstrap replicate index.

  • Delta_Cost, Delta_Effect, ICER, Mean_Cost_g1, Mean_Cost_g2, Mean_Effect_g1, Mean_Effect_g2, NMB.

Examples

data(cua_base)
res <- icers_base(
  data = cua_base,
  group = "group",
  cost = "cost_total",
  effect = "effect",
  R = 100,
  lambda = 25000,
  subgroup_vars = c("diabetes", "HTA")
)
df_combined <- combine_icers_results(res)
